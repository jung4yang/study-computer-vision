## 컴퓨터 비전의 지형도 (먼저 전체 그림)

CV는 핸드트래킹 하나가 아니라 넓은 분야예요. 6개월 동안 아래를 **두루 건드려보고**, 어느 게 재밌는지 스스로 발견하는 걸 목표로 합니다.

| 대분류 | 대표 태스크 | 예시 |
|--------|------------|------|
| **인식/분류** | Image Classification | 사진이 개냐 고양이냐 |
| **검출** | Object Detection | 이미지 속 물체 위치+종류 (YOLO) |
| **분할** | Segmentation | 픽셀 단위로 영역 나누기 (U-Net, SAM) |
| **추적** | Tracking | 영상에서 물체를 프레임 간 따라가기 |
| **자세/키포인트** | Pose/Landmark | 손·몸·얼굴 관절 추정 (MediaPipe) |
| **3D/깊이** | Depth, SLAM, NeRF | 단안 깊이추정, 3D 복원, 공간 인식 |
| **영상 이해** | Video, Action Recognition | 행동 인식, 시간 정보 |
| **생성/편집** | Generative | Diffusion, 이미지 생성·편집·복원 |
| **멀티모달** | Vision-Language | CLIP, 이미지+텍스트 (요즘 대세) |

> 이 표가 앞으로의 나침반이에요. 각 단계에서 여기 여러 칸을 실제로 만져봅니다.

## 1단계 — 기초 다지기 (1~4주차)

수학은 "시험용"이 아니라 "논문·강의를 알아듣기 위한" 최소한만. 완벽주의 금물.

**1~2주차: Python/NumPy + 수학 워밍업**
- NumPy 배열·행렬 연산 (이미지=배열)
- 선형대수: 벡터·행렬·행렬곱 → 3Blue1Brown "Essence of Linear Algebra"
- 미적분: 미분·기울기·연쇄법칙 → 3Blue1Brown "Essence of Calculus"
- 확률: 평균·분산·분포 기본

**3~4주차: 전통 이미지 처리 (OpenCV)**
- 읽기/자르기/리사이즈, 색공간, 필터·블러, 엣지(Canny), 윤곽선, 히스토그램
- **미니 프로젝트 ①**: 자유 주제 (예: 색/얼굴 검출, 문서 스캐너, 간단한 파노라마 스티칭)

✅ 체크포인트: "이미지는 (H×W×C) 숫자 배열"이라는 게 코드로 손에 익었는가?

## 2단계 — 딥러닝과 CNN 핵심 (5~12주차)

여기가 모든 현대 CV의 공통 토대. 시간을 가장 많이 투자.

**5~6주차: 신경망 기초**
- 순전파/역전파, 손실함수, 경사하강법, 최적화
- 자료: 3Blue1Brown "Neural Networks" + Andrew Ng Deep Learning Specialization(청강 무료)
- PyTorch 시작: 공식 "60 Minute Blitz"

**7~9주차: CNN**
- 합성곱·풀링·수용영역, LeNet→AlexNet→VGG→ResNet 흐름
- **메인 강의: Stanford CS231n (2025)** — CV 딥러닝 정석. 유튜브 강의 + cs231n.github.io 노트. CNN·Transformer·Diffusion·Vision-Language까지 최신 커버
- **프로젝트 ②**: MNIST → CIFAR-10 분류를 PyTorch로 밑바닥부터

**10~12주차: 전이학습 + 실전 학습 기법**
- 사전학습 모델 fine-tuning, 데이터 증강, 과적합/정규화, 배치정규화, 학습률 스케줄
- **프로젝트 ③**: 직접 모은 데이터로 분류기. 남과 다른 데이터셋이 포트폴리오에서 눈에 띔

✅ 체크포인트: 새 분류 문제를 데이터 준비→학습→평가까지 혼자 코드로 짤 수 있는가?

## 3단계 — CV 전 분야 순회 (13~20주차)

지형도의 여러 칸을 **직접 돌려봅니다.** 각 2주 미만으로 빠르게 훑되, 코드는 꼭 돌려서 "느낌"을 잡기. 그러다 꽂히는 분야가 생기면 그걸 간판 프로젝트로.

**13~14주차: 객체 탐지 (Detection)**
- 바운딩 박스, IoU, mAP, 1-stage vs 2-stage
- 실전: Ultralytics **YOLO**로 커스텀 데이터 라벨링→학습→추론 (라벨링: Roboflow/CVAT)

**15주차: 분할 (Segmentation)**
- Semantic vs Instance, U-Net, Mask R-CNN, **SAM(Segment Anything)** 데모

**16주차: 추적 + 자세추정 (Tracking & Pose)**
- 영상 물체 추적(ByteTrack 등), 자세추정(MediaPipe Pose) — 당신이 이미 강점 가진 영역, 빠르게 확장

**17주차: 3D / 깊이 (Depth & 3D)**
- 단안 깊이추정(Depth Anything 등), 스테레오, NeRF·3D Gaussian Splatting 개념 맛보기

**18주차: 생성·멀티모달 (Generative & Vision-Language)**
- Diffusion 기반 이미지 생성·편집 개념, **CLIP**과 Vision-Language 모델 (요즘 필수 교양)

**19~20주차: 간판 프로젝트 완성 (엔드투엔드)**
- 위에서 제일 끌린 분야로 **하나를 제대로**: 데이터 → 학습 → 성능 개선 → 데모(Streamlit/Gradio 웹앱) → 결과 정리
- 여기서 HCI 배경을 살리고 싶으면 자연스럽게 인터랙티브 데모로 만들어도 좋지만, **필수는 아님.** 순수 인식·검출 프로젝트도 훌륭한 포트폴리오

✅ 체크포인트: CV의 여러 태스크 중 최소 4~5개를 "코드로 돌려본" 경험이 있고, 그중 하나는 깊게 팠는가?

## 4단계 — 포트폴리오 & 취업 (21~26주차)

**21~22주차: 포트폴리오 정리**
- GitHub 정돈: 프로젝트마다 깔끔한 README(문제·데이터·방법·결과·데모). 잘 정리된 3~4개 > 지저분한 10개
- 대표 프로젝트 1개는 블로그 글로 (시행착오·설계 결정 → 사고력 증명)

**23~24주차: 공학 기본기 + 면접 대비**
- 국내 취업이면 코딩테스트 병행(하루 1시간, 자료구조·알고리즘)
- 기술면접 단골: CNN 원리, 과적합 대처, 전이학습, 평가지표(mAP/IoU), 모델 선택 이유, 실시간·경량화(ONNX, 지연시간)

**25~26주차: 지원 & 실전**
- 이력서: 프로젝트 성과를 숫자로 ("정확도 82%→91%", "추론 속도 2배")
- 채용 공고 역분석: 지원처가 요구하는 스킬(PyTorch/TensorFlow, C++, Docker, 클라우드, TensorRT 등) 파악해 부족분 보충
- 링크드인·포트폴리오 정비 후 지원 시작

✅ 최종 체크포인트: GitHub 링크 하나로 "이 사람 CV 제대로 할 줄 아네"가 전달되는가?

## 세부 진로는 언제 정하나

지금 정할 필요 없어요. 3단계에서 여러 분야를 만져보면 자연스럽게 갈립니다:
- **검출·분할·추적이 재밌다** → 산업용 비전, 자율주행, 로보틱스
- **3D·깊이·SLAM이 끌린다** → AR/VR, 자율주행, 3D 복원
- **생성·멀티모달이 좋다** → 생성 AI, 비전-언어 모델 연구·응용
- **자세·인터랙션이 여전히 좋다** → HCI/AR 인터랙션 (당신의 원래 강점, 옵션으로 열어둠)

어느 쪽이든 **2단계까지의 토대는 완전히 공통**이라, 지금은 넓게 가는 게 정답이에요. 방향이 잡히면 그때 심화 자료를 다시 짜드릴게요.
